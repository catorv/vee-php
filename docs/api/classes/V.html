makeUrl()

makeUrl(string $action, string $controller, array $addList, array $removeList) : string

生成一个当前页面的URL (所传入的参数只用于生成URL，并不真正更新 $uri['arguments'] 与 $_REQUEST 的值)

本函数接受两种参数调用方式：

1. fn($action, $controller, $module, $addList, $removeList)
2. fn($addList, $removeList)

Parameters

string $action

修改动作名称
string $controller

修改控制器名称
array $addList

增加/修改参数项(包括 $uri['arguments'] 与 $_REQUEST)
array $removeList

删除参数项(包括 $uri['arguments'] 与 $_REQUEST)

Returns

string
